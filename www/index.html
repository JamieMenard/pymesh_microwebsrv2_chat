DIRECTIONS:

Click on the LHVW Chat link below, wait 30 seconds or so for the "Welcome" Message to load and send messsages!

This is more like HAM radio than Gchat, so don't expect that all the messages get to where you expect. If you don't get an answer
Wait a bit and try to resend!
And even if the "house" you want to talk to is listed as being online, that doesn't mean
there's a computer connected to it! The message will be saved to the node and they can access it later.

If you want to see a full list of messages to your node since it's been powered on, click the "Chat Log" link towards the bottom of
of the page. If should have any messages saved that you missed while not having a computer connected.
Be sure to refresh the chat log page to see the lastest messages.
Also, anytime the node loses power or needs a restart, go to the configuration page and click submit without changing any values.
This'll reset the internal clock
